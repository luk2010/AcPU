AcPU

Atlanti's Corp Php Utility

Download Download fork AcPU on GitHub

Atlanti's Corp Php Utility is a fantastic tool that help you develop cool web page in minute, without thinking about every bugs you may have using html, BDD, or JavaScript. It is designed for programmers who already used any compiled language like C, C++, etc, because it use a Oriented-Object interface. For designers, you also can import directly your Html Code to the AcPU HtmlElements system.

The basic code to draw a simple page is :

<?php

require 'AcPU.php'; // This depends on where you stores AcPU

$htmlConstructor = AcPU::get()->createHTMLConstructor('My Constructor');

$htmlConstructor->getCenter()->createChildWithText('h1', 'A big title !', 'My Big Title');
$htmlConstructor->getCenter()->addParagraphe('Hello World !');

$page = Page::get();
$page->setHTMLConstructor($htmlConstructor);
$page->setManualHeader(false);

$page->addMeta('charset', 'utf-8');

$page->draw('My Pretty page', '');

?>

As you can see, beginning takes just a few more lines, but then you can quickly add a new element, with just one explicit line. You also can create your own element, to fully customize your AcPU.
